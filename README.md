# Portfolio  - Next.js
Next.js를 사용하여 만든 포트폴리오입니다.  
[바로가기](https://olive23oct.github.io/)

##  개발환경 세팅 
1. create new repository
2. ```bash
    #프로젝트 생성
    $ npx create-next-app
    #프로젝트명 기입(대문자사용불가)
    ```
3. _document.js 추가 (부트스트랩)
4. ```bash
    yarn build
    ```
5. package.json 추가
    ```json
     "export": "next export"
    ```
6.  ```bash
     yarn export
    ``` 
7. out 폴더 생성 -> 상위 디렉토리로 내용물 이동
